# Capitolul 3: Materiale și Metodologie Experimentală (Actualizat)

## 3.1 Designul celor 7 Rețete Experimentale (R0-R6)

Programul experimental este structurat pentru a izola și a analiza contribuția fiecărui component:

| Rețetă | Compoziție | Rol Experimental |
| :--- | :--- | :--- |
| **R0** | Beton Martor | Etalon (fără fibre, fără cenușă) |
| **R1** | Beton + Fibre Polimerice | Influența individuală a fibrelor PP |
| **R2** | Beton + 18% Cenușă Zburătoare | Influența substituției cimentului |
| **R3** | Beton + Fibre de Sticlă | Influența individuală a fibrelor AR-Glass |
| **R4** | Beton + Cenușă + Fibre Polimerice | Sistem hibrid binar |
| **R5** | Beton + Fibre Sticlă + 20% Cenușă | Sistem hibrid binar optimizat |
| **R6** | Beton + Sticlă + Polimerice + Cenușă | **Sistem hibrid complex (Ternar)** |

## 3.2 Materiale utilizate
- **Ciment:** CEM I 42.5R.
- **Cenușă:** Class F (10-25% substituție).
- **Fibre Sticlă AR:** 1.0-2.5 kg/m³.
- **Fibre PP:** 1.0-1.5 kg/m³.

## 3.3 Procedura de testare
- Rezistențe la compresiune la 7 și 28 de zile.
- Rezistențe la încovoiere pe prisme.
- Absorbția de apă pentru evaluarea durabilității.
